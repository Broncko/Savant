Class Twig_Loader_Array

Description
Description | Vars (details) | Methods (details)

Implements interfaces:

  • Twig_LoaderInterface

Loads a template from an array.

When using this loader with a cache mehcanism, you should know that a new cache key is generated each time a template content "changes" (the cache key being the source code of the template). If you don't want to see your cache grows out of control, you need to take care of clearing the old cache file by yourself.
